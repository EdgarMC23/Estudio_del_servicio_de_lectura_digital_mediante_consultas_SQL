El objetivo de este proyecto fue analizar una base de datos de libros, editoriales, autores y reseñas para entender el comportamiento de los amantes de la lectura y proponer mejoras en servicios o crear nuevos productos. Para lograrlo, se utilizó Python con las librerías Pandas y SQLAlchemy para conectar y consultar una base de datos PostgreSQL. Se realizó un análisis exploratorio de datos que reveló información valiosa, como el gran número de libros publicados después del año 2000 (819), la identificación de editoriales líderes como Penguin Books, y autores altamente valorados como Diana Gabaldon. Además, se observó el alto compromiso de usuarios activos, quienes promedian más de 24 reseñas. Estos hallazgos permitieron formular una sólida propuesta de valor para un nuevo servicio, incluyendo un catálogo en constante crecimiento, un algoritmo de recomendación avanzado y un espacio dedicado a la interacción de la comunidad de lectores.
